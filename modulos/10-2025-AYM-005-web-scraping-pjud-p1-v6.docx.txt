	Scraping (Selenium, Playwright, Puppeteer, u otro)
	Código abierto
	Sin costo de licencia. Costos asociados a configuración, desarrollo y mayor consumo de recursos de servidor.
	Resolución de CAPTCHA (ej: 2Captcha, AntiCaptcha, AI propia)
	Servicio propietario
	Servicios de terceros cobran por solicitud resuelta. Alternativa: desarrollo propio con IA (alto costo inicial, evita gasto recurrente).
	Lenguajes Backend (Python, Node.js)
	Código abierto
	Gratuito. Costos asociados solo al desarrollo e infraestructura (ej. hosting, workers).
	Base de datos (PostgreSQL / MySQL)
	Código abierto
	Sin costo de licencia. Solo costos de hosting/infraestructura.
	Servidor / Infraestructura (ej: AWS, GCP,Azure, VPS local)
	Propietario
	Pago según uso de CPU/RAM/almacenamiento. Escalable según volumen de causas consultadas.
	Dashboard Web (React, Vue, Angular, o HTML/CSS/JS)
	Código abierto
	Gratuito. Solo costos de desarrollo e integración con backend y base de datos.
	Seguridad y cifrado (Vault, JWT, bcrypt, TLS/SSL)
	Código abierto
	Sin costo de licencia. Costos asociados a certificados SSL y configuración.
	Notificaciones (correo, push)
	Mixto
	Email: puede usarse SMTP gratuito (limitado) o servicios pagos (Sendgrid, Amazon SES). Push/websocket: open source, costo de servidor.
	Scheduler / Workers (Celery, CRON, RabbitMQ, Redis)
	Código abierto
	Gratuito. Costos asociados al servidor que mantenga las tareas en ejecución.
	Exportación (PDF, Excel – librerías tipo Pandas, ReportLab, SheetJS)
	Código abierto
	Gratuito. Costos asociados a desarrollo y rendimiento en generación de reportes masivos.
	El sistema no incluye gestión de suscripciones ni planes de pago, lo que permite optimizar costos al enfocarse solo en los módulos esenciales de scraping, base de datos y panel web (conversable para App Mobile Android/iOS).
